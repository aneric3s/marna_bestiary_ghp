**Класс брони:** 16 (нестабильная реальность)

**Хиты:** 85 (9к10 + 36) — _вариация от 70 до 115_

**Описание внешности:** Сгусток абсолютной тьмы размером с человека, в котором иногда проступают очертания чего-то знакомого — лица, руки, предмета, знакомого пейзажа. Вокруг него — зона тишины, где не слышно ни звука. Он не издает шума при движении, не оставляет следов. Касание его стирает кусочки реальности.

**Среда обитания:** Места, где Ветвь особенно больна, трещины в коре Марны, древние руины, места великих трагедий.

**Сопротивления и уязвимости:**

| Тип урона     | Степень             |
| ------------- | ------------------- |
| Колющий       | Иммунитет           |
| Режущий       | Иммунитет           |
| Дробящий      | Сопротивление (1/2) |
| Огнестрельный | Сопротивление (1/2) |
| Проникающий   | Иммунитет           |
| Пробивающий   | -                   |
| Огонь         | -                   |
| Холод         | -                   |
| Кислота       | Иммунитет           |

| Психика       | Иммунитет           |

| Свет          | Уязвимость (1/2)    |
| Серебро       | Уязвимость (1/2)    |

**Скорость:** 6 клеток (летает)

**Атаки:**

1. **Касание забвения** _(ближняя, 1 клетка)_  
    _Меткий бросок:_ +8 к попаданию  
    _Урон:_ 3к8 некрозом + стирание памяти  
    _Особенность:_ При попадании цель теряет 1к6 Интеллекта или Мудрости (восстанавливается за длинный отдых). Если характеристика падает до 0 — цель впадает в кататонию.
    
2. **Взгляд Пустоты** _(дальняя, 6 клеток)_  
    _Меткий бросок:_ спасбросок Мудрости Сл 16  
    _Урон:_ 3к10 психикой  
    _Особенность:_ При провале цель также теряет 1к4 Харизмы (стирается личность).
    
3. **Зона тишины (действие, перезарядка 1к4 хода)**  
    _Меткий бросок:_ область радиусом 3 клетки  
    _Урон:_ нет  
    _Особенность:_ В области не работает магия с вербальными компонентами, не слышно звуков. Существа внутри получают 2к6 некроза в начале хода.
    

**Способности:**

1. **Невидимость в тишине (пассивно)**  
    В зоне тишины или в полной темноте Безгласный невидим. Его можно заметить только по искажению реальности (Восприятие Сл 17).
    
2. **Пожиратель воспоминаний (бонусное действие)**  
    Безгласный может съесть воспоминание цели, которая уже потеряла Интеллект или Мудрость от его атак. Цель забывает последний час (или важное событие по желанию Мастера).
    
3. **Искажение реальности (реакция)**  
    Когда по Безгласному попадают, он может исказить пространство. 50% шанс, что атака пройдет мимо (телепортация на 1 клетку).
    
4. **Крик Пустоты (действие, 1/день)**  
    _Меткий бросок:_ спасбросок Телосложения Сл 17 для всех в радиусе 5 клеток  
    _Урон:_ 5к10 некроза  
    _Особенность:_ Безгласный издает беззвучный крик, разрывающий реальность.
