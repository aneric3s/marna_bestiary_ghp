**Класс брони:** 14 (скользкая)

**Хиты:** 37 (5к8 + 15) — _вариация от 30 до 50_

**Описание внешности:** Мелкое, тощее существо ростом с ребенка, похожее на старуху, поросшую тиной, ряской и болотной травой. Кожа зеленоватая, морщинистая, пальцы длинные, с когтями. Волосы — спутанные водоросли. Глаза желтые, светятся в темноте. Одежда — лохмотья из тины. Передвигается бесшумно, иногда прыгает, как лягушка. Пахнет болотом и гнилью.

**Среда обитания:** Болота, топи, мангровые заросли, торфяники, старые трясины.

**Сопротивления и уязвимости:**

| Тип урона     | Степень                            |
| ------------- | ---------------------------------- |
| Колющий       | -                                  |
| Режущий       | -                                  |
| Дробящий      | -                                  |
| Огнестрельный | -                                  |
| Проникающий   | -                                  |
| Пробивающий   | -                                  |
| Огонь         | Уязвимость (1/2)                   |
| Холод         | -                                  |
| Кислота       | Сопротивление (1/2)                |

| Некроз        | -                                  |

| Железо        | Уязвимость (1/2) (холодное железо) |

**Скорость:** 5 клеток (по земле), 4 клетки (по воде)

**Атаки:**

1. **Грязные когти** _(ближняя, 1 клетка)_  
    _Меткий бросок:_ +6 к попаданию  
    _Урон:_ 2к4 + 2 режущий  
    _Особенность:_ Грязные, могут занести инфекцию (спасбросок Телосложения Сл 12 или болезнь).
    
2. **Укус** _(ближняя, 1 клетка)_  
    _Меткий бросок:_ +5 к попаданию  
    _Урон:_ 1к6 + 2 колющий  
    _Особенность:_ Зубы острые, гнилые.
    
3. **Призыв трясины (действие, перезарядка 1к4 хода)**  
    _Меткий бросок:_ спасбросок Ловкости Сл 13 для одной цели в пределах 5 клеток  
    _Урон:_ нет  
    _Особенность:_ Под целью разверзается трясина. Цель должна совершить спасбросок или провалиться по пояс (скорость 0, действие на освобождение).
    

**Способности:**

1. **Маскировка**  
    В болоте кикимору почти не видно (Восприятие Сл 16). Она может сливаться с кочками и тиной.
    
2. **Мимикрия голоса**  
    Кикимора может подражать голосам: плачу детей, крикам о помощи, голосам друзей. Проверка Мудрости Сл 14, чтобы распознать обман.
    
3. **Повелительница тумана**  
    Кикимора может наслать легкий туман в радиусе 10 клеток, который дает легкое укрытие всем (помеха на дальние атаки).
    
4. **Слабость к железу**  
    Холодное железо обжигает кикимору. При попадании железным оружием она получает дополнительные 1к4 урона и должна совершить спасбросок или закричать (выдает позицию).
