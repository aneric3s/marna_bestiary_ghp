**Класс брони:** 14 (известковый панцирь)

**Хиты:** 15 (3к6 + 6) — _вариация от 12 до 24_

**Описание внешности:** Небольшое, похожее на актинию существо, прикрепленное ко дну или камню. Тело мягкое, студенистое, покрытое тонким известковым панцирем. Сверху — множество щупалец, которыми оно фильтрует воду. Цвета — от белого до розового и оранжевого. Медленно покачивается в воде. Колонии таких существ создают целые рифы.

**Среда обитания:** Мелководные рифы, теплые моря, подводные скалы, территории Моргиров.

**Сопротивления и уязвимости:**

| Тип урона     | Степень             |
| ------------- | ------------------- |
| Колющий       | Сопротивление (1/2) |
| Режущий       | -                   |
| Дробящий      | Уязвимость (1/2)    |
| Огнестрельный | -                   |
| Проникающий   | Сопротивление (1/2) |
| Пробивающий   | -                   |
| Огонь         | Уязвимость (1/2)    |
| Холод         | -                   |
| Кислота       | Уязвимость (1/2)    |
| Яд            | Иммунитет           |
| Психика       | Иммунитет           |
| Некроз        | -                   |
| Свет          | -                   |

**Скорость:** 0 клеток (прикреплен)

**Атаки:**

1. **Стрекательные щупальца** _(ближняя, 1 клетка)_  
    _Меткий бросок:_ +3 к попаданию  
    _Урон:_ 1к4 + паралич  
    _Особенность:_ При попадании цель должна совершить спасбросок Телосложения Сл 11. При провале — парализована на 1 ход (щупальца впрыснули нейротоксин).
    
2. **Известковый плевок** _(дальняя, 2 клетки)_  
    _Меткий бросок:_ +3 к попаданию  
    _Урон:_ 1к3 дробящий  
    _Особенность:_ Плюется мелкими известковыми шариками. Использует редко.
    

**Способности:**

1. **Строитель рифа**  
    Кораллит выделяет известковый секрет, который затвердевает в воде. За год колония может создать целый риф. За час — небольшое укрытие.
    
2. **Колония**  
    Кораллиты живут колониями (2к6 особей на одной скале). Если атаковать одного, соседи могут выпустить облако известковой мути (застилает обзор на 1 ход).
    
3. **Лечебный секрет**  
    Секрет кораллита обладает слабыми антисептическими свойствами. Если приложить кусочек к ране, остановит кровотечение и даст +1 к следующему спасброску от заражения.
    
4. **Симбиоз**  
    Вокруг кораллитов всегда водится мелкая рыбешка. Если кораллит в безопасности, можно поймать 1к4 мелких рыбок для еды.
